orthography
  • ch h → x
  • ó → о̄
  • ą → ǫ
  • e (movable) → ĕ
  • rz → rj
  • l, ł → lj, l
  • ń, ś, ź → nj, sj, zj
  • ć → tj
  • ci, ci(V) → tji, tj
  • dzi, dzi(V) → dji, dj
  • i(V) → j
  • je → ě
  • cz → ḳ, ṫ
  • sz → ẋ, ṡ
  • ż → ġ, ż
  • y → ı
  • ji → i

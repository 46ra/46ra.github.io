cantonese

i introduce yet another pinyin system for cantonese. it is consistent and compact, though it may seem complex at zeroth glance. the system has an ASCII version in addition to an original Unicode one, so that we can use it every where.

the author may make incompatible changes in the system.

the author designed the system just to suit vis preference or aethetics, and force no one to use it. if some one, however, would like it and try to use it, ve would be very glad.
